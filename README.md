# HC

Hierarchical Clustering in `python3`, (used my library [Matrix7](https://github.com/LogX7/Matrix7))

## requirements

```bash
pip install matplotlib scipy numpy
```

## usage

```bash
python main.py
```

### example

- input :

```python
# 2 coordinates, 6 samples
I = [
    [1, 1],
    [1, 2],
    [1, 6],
    [1, 8],
    [1, 9],
    [2, 1],
]
```
